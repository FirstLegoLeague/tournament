How to design a page?

When you start thinking about how a page should look, you need to keep a couple of things in mind:

  • What will the page look like when using screens of different sizes? Whic screen sizes should this screen fit?
  • What is the flow of actions in this screen? UX experts tell us that the flow should of action should follow the flow of reading - from left to right, and from top to bottom.
  • If the screen has many object, maybe it would be better to split it into several screens, i order to not overwhelm the user.

So when should I use each layout component?

  • Use grid a lot, it helps devide the page easier.
  • Use menu a lot, it is the basic element for actions, and very helps to organize.
  • Use tabs to organize your pages easily.
  • Use topbar when you need to have a similar look in all pages, with the same menus and actions.
  • Use secondary topbar if you use the topbar and want stuff to change between pages. The secondary topbar is meant to have all the secondary actions.
  • Use off-canvas only when the usage of its content is quick, like accessing different pages. Don't use it, for example, for complex query building. In this cases, you should use grid together with toggler.
